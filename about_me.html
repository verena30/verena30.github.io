About Me?

Disini saya akan sedikit menceritakan kepada Anda tentang diri saya

Simak informasi berikut tentang diri saya

1. Keseharian

Untuk saat ini saya masih berstatus sebagai pelajar dan sedang berada di bangku sekolah yaitu SMA. Jadi untuk keseharian saya biasanya saya bersekolah. Namun pada akhir pekan biasanya saya akan mengikuti kegiatan-kegiatan lain di organisasi.

2. Hobi

Hobi saya adalah menyanyi. Bagi saya menyanyi adalah hal yang menyenangkan. Menyanyi terkadang membuat saya menjadi merasa senang dan merasa tenang. Karena saya suka bernyanyi maka saya juga suka mendengarkan musik atau lagu. Bagi saya musik atau lagu adalah teman saya yang selalu menemani saya disetiap kegiatan yang saya lakukan.

3. Organisasi

Sejak saya SMP saya mulai aktif dalam kegiatan berorganisasi. Contoh organisasi yang saya ikuti adalah OSIS. Saya menjadi anggota OSIS di SMP dan di SMA yaitu saat ini saya juga masih menjadi bagian dari pengurus OSIS. Tidak hanya OSIS saya juga mengikuti berbagai macam organisasi lain di luar sekolah.
